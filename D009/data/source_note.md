# 데이터 출처 노트

- **데이터셋 이름**: KBO 팬 멤버십/구매 데이터 (`kbo_fan_data_sample.csv`)
- **출처**: Kaggle이나 공공데이터포털에서 받은 데이터가 아닙니다. 커리큘럼 실습을 위해 제공된
  가상(synthetic) 샘플 데이터로, 결측치·중복·이상치·날짜 표기 혼재·범주 표기 혼재가 의도적으로
  섞여 있는 연습용 데이터셋입니다. 공개 URL은 없습니다.
- **이 데이터를 선택한 이유**: 빵셀러 예시 데이터 대신, 이번 과제 가이드의 "선택 2. 외부 데이터
  사용하기" 흐름을 연습하기 위해 실제 도메인(프로 스포츠 팬 멤버십)을 가진 별도 데이터셋으로
  진행했습니다. 범주형(지역, 구단, 멤버십 등급)과 수치형(나이, 지출액, 관람 경기수) 컬럼이 함께
  있고, 날짜 컬럼 2개, 표기 혼재·결측·중복·이상치가 모두 존재해 과제 가이드의 선정 기준
  ("결측치·중복·이상치·자료형·표기 혼재 중 2가지 이상 점검 가능")을 충족합니다.
- **개인정보 처리**: 원본 파일에는 `email` 컬럼이 있었으나, 과제 가이드의 "개인정보 포함 데이터
  사용 금지" 항목에 따라 분석 시작 전 컬럼 자체를 제거했습니다. `customer_id`는 `KBOFAN0000001`
  형식의 합성 식별자로 실제 개인을 특정할 수 없어 그대로 사용했습니다. 이 폴더(`data/`)에 있는
  `kbo_fan_data_sample.csv`도 email 컬럼이 제거된 버전입니다.
- **행의 의미(관측 단위)**: 한 행 = 한 명의 KBO 팬(멤버십 가입 고객)
- **전체 크기**: 5,075행 × 14열 (email 제거 후 기준)
